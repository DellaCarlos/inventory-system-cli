# Terminal-Based Inventory Management

A simple **terminal-based inventory management system** built with **Golang**, focused on learning, clean logic, and basic Go best practices.

## Features

- List inventory items  
- Add new items  
- Remove items by index  
- Data persistence (inventory saving)  
- Fully terminal-based interface  

## Technologies Used

- **Go (Golang)**
- Standard input/output (CLI)
- Slices and functions
- File handling for data persistence
